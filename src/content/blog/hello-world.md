This is the first post on the new site. To write a new one, add a Markdown file to `src/content/blog/`, give it a `title` and `date` in the frontmatter, and push — the GitHub Actions workflow builds and deploys it automatically.

No build step to run locally unless you want to preview with `npm run dev` first.
